# WS1 Product & Domain PRD Path

## Scope
- Business capabilities and rules for workspace, project, work item, cycle, module, page.

## Deliverables
- User stories with Given/When/Then.
- Entity state machines (work item state transitions).
- Role-permission matrix.
- Acceptance criteria by feature.

## Exit Criteria
- No ambiguous role behavior.
- Every feature has measurable acceptance criteria.
